#  Scientific Signal Processing: Noise Filtering & Detector Calibration

> Инструменты для обработки экспериментальных данных: фильтрация шумов, расчёт корреляций $g^{(2)}$, калибровка квантовой эффективности детекторов.  
> Демонстрирует навыки работы с реальными зашумленными данными — применимо в IoT, телекоме, финтехе (детекция аномалий).

##  Для чего это может быть полезно в Data Science
- **Работа с зашумленными данными**: методы фильтрации, выбросы, артефакты измерений
- **Временные ряды**: анализ корреляций, стационарность, детрендинг
- **Калибровка моделей**: перенос методов калибровки детекторов на калибровку ML-моделей
- **Интерпретируемость**: физическая природа признаков помогает объяснять предсказания

##  Стек
- **Язык**: Python 3.x
- **Библиотеки**: `numpy`, `pandas`, `scipy`, `matplotlib`
- **Формат**: Jupyter Notebook для анализа
